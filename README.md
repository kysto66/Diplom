1)Создать документ "Обслуживание клиента"
2)Заполнить данные документа "Обслуживание клиента"
3)Назначить роли сотрудникам по специальности:
-роль "Бухгалтер IT фирмы"
-роль "Кадровик-расчетчик"
-роль "Менеджер"
-роль "Специалист"
4)Создать Вид договора "Абонентская плата"
5)Заполнить константу "Токен управления телеграм-ботом"
6)Заполнить константу "Идентификатор группы для оповещения"
7)Заполнить константу "Номенклатура Абонентская Плата"
8)Заполнить константу "Номенклатура Работы Специалиста"
9)Заполнить константу "Размер НДФЛ"
10)Создать записи в регистре сведений "Условия оплаты сотрудников"
11)Создать записи в регистре сведений "Фиксированная премия" 
12)Создать документ "Начисление зарплаты"
13)Заполнить данные документа "Начисление зарплаты"
14)Создать документ "Начисление фиксированной премии"
15)Заполнить данные документа "Начисление фиксированной премии"
16)Создать документ "Выплаты зарплаты"
17)Заполнить данные документа "Выплаты зарплаты"
18)Заполнить регистр сведений "График работы" используя обработку "Заполнения графика" , выбрав период и выходные дни
19)Создать документ "График отпусков"
20)Заполнить данные документы "График отпусков"
